# MiseEnLabelPrivacy

Privacy Policy landing page for the MiseEnLabel app.

## About

This repository hosts the privacy policy for the MiseEnLabel application. The privacy policy explains how user data is collected, used, and protected.

## View the Privacy Policy

Visit the [Privacy Policy page](https://sandwitch-maker.github.io/MiseEnLabelPrivacy/) to read the full policy.

## Local Development

To view the page locally:

```bash
# Clone the repository
git clone https://github.com/sandwitch-maker/MiseEnLabelPrivacy.git
cd MiseEnLabelPrivacy

# Open index.html in your browser
# Or start a local server
python3 -m http.server 8080
# Then visit http://localhost:8080
```
